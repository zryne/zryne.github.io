Vim Quick Reference

Find :/<word>
n
Shft+n		 Highlight all "word" in the current file.
Go to next "word".
Go to previous "word".
:/^$ Highlight all empty lines.
Substitution :%s/\s\+/\r/g Replace spaces (one or contiguous) with newlines in the whole file.
:%s/\s\+/\r/gc Replace spaces (one or contiguous) with newlines in the whole file, confirming each.
:g/red/s/blue/green Replace "blue" with "green" in lines that contain "red".
:g!/red/s/blue/green Replace "blue" with "green" in lines that do NOT contain "red".
~ Change the character's case under the cursor.
:g/^$/d Delete all empty lines.
File Handling :split <file>
ctrl+w		 Open another file named "file" in split window.
Toggle between the two.
gf Edit the file under the cursor.
:r! <cmd> Execute the <cmd> and insert output at the cursor.
Sorting :!sort Sort (in ascending order) all lines in the current file.
:2,7!sort Sort (in ascending order) only lines 2-7 in the current file.
